# python-web-bottle-mvc-ssl-hello-world

## Description
Uses MVC to render html pages with self-signed ssl.

## Tech stack
- python
  - bottle
- openssl

## Docker stack
- python:latest
- alpine:edge

## To run
`sudo ./install.sh -u`
- [Available](https://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
